# PRODUCT SUMMARY

VIRGIL is an AI-powered payroll integrity and proof-of-life verification system that detects ghost workers before salary payments are released.

The system:
* analyzes payroll datasets
* checks proof-of-life signals
* detects suspicious anomalies
* risk scores workers
* blocks suspicious salary disbursement
* releases verified payments through Squad API

Core principle:
"When confidence is low, VIRGIL does not pay."

# PROOF-OF-LIFE SYSTEM

Create worker verification flow.
Workers DO NOT use admin dashboard.

Worker flow:
1. Open verification link
2. Enter Staff ID
3. Enter OTP
4. Complete selfie/liveness check
5. Complete random prompt
6. Receive verification result

Random prompts:
* Blink twice
* Turn left
* Smile
* Raise right hand

Record:
* device info
* timestamp
* verification confidence
* location if allowed
* liveness result

Output:
* VERIFIED
* NEEDS REVIEW
* FLAGGED

For demo: simulate face/liveness scoring realistically.
